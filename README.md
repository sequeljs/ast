# Sequel.js - AST

```JavaScript
npm install --save @sequeljs/ast
```

## Inspiration

`@sequeljs/ast` is basically a rewrite of
[Rails' Arel](https://github.com/rails/rails/tree/v6.0.3.2/activerecord/lib/arel)
library in TypeScript.
